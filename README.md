## Outfit Store
Ecommerce model 100% functional 


## Technology 

Here are the technologies used in this project.

* Next JS
* React JS
* Mongo DB
* TailwindCSS
* Paypal

## Services Used

* Github
* Vercel
* Mongo DB Atlas

## Getting started

* Dependency
  - Formik
  - JS-cookie
  - Typescript
  - Toastfy
  - React-icons
  - YUP
  - Mongoose
  - Next-auth
  - Axios
  
* To install the dependencies.
```bash
  $ npm install
  ```
  
* To run the project.
```bash
  $ npm run start
  ```
