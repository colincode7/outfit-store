## Getting started

* Dependency
  - Formik
  - JS-cookie
  - Typescript
  - Toastfy
  - React-icons
  - YUP
  - Mongoose
  - Next-auth
  - Axios
  
* To install the dependencies.
```bash
  $ npm install
  ```
  
* To run the project.
```bash
  $ npm run start
  ```
